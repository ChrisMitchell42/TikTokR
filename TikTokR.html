Exercise 4: OBJECTS

Now that we know how R can classify some basic types of information, we will now learn how to store them. R allows you to save data by storing it inside an R object. What is an object? An object is just a name that you can use to call up stored data. For example, you can save data into an object. So for example if you‚Äôre given raw data about different colored cars, you can classify them according to their color. Wherever R encounters the object, it will replace it with the data saved inside.

To create an object, choose a name and then use the < symbol, followed by a minus - sign to save data into it. This should look like <-. R will then make an object, give it your name, and store in it whatever follows the arrow.
