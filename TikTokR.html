Welcome to your first lesson in R. What is R you may ask? R is a language that allows you to conduct statistics and data analysis. R gives you a way to talk to your computer. R is more of a tool to understand data than a programming language used to build software applications for e.g. like Python. You may wonder why we are teaching you to code and this is simply because it is an employable skill that will be beneficial throughout your degree and in the future (many employers look for this skill in their graduate applicants).

Exercise 3: TYPES OF DATA

In R and programming, data types are the classifications we give different kinds of information pieces. The different types of data we will be working with throughout this module are:

  • Numeric –> Any number with or without a decimal point
  • Character –> Any grouping of characters on your keyboard or text. These strings (what they’re referred to in R) are surrounded by single quotes ‘…’ or double quotes “…”
  • Logical –> This type of data has only two possible values TRUE or FALSE (without the quotes)
  • Vectors –> List of related data that is all the same type
  • NA –> This type of data represents the absence of a value and is represented by the keyword NA. There can be numeric NA, character NA and a logical NA.

Exercise 4: OBJECTS

Now that we know how R can classify some basic types of information, we will now learn how to store them. R allows you to save data by storing it inside an R object. What is an object? An object is just a name that you can use to call up stored data. For example, you can save data into an object. So for example if you’re given raw data about different colored cars, you can classify them according to their color. Wherever R encounters the object, it will replace it with the data saved inside.

To create an object, choose a name and then use the < symbol, followed by a minus - sign to save data into it. This should look like <-. R will then make an object, give it your name, and store in it whatever follows the arrow.
